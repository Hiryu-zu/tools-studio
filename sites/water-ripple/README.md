# 水面リップル背景エフェクト

マウス操作で波紋が広がるWebGL背景エフェクトのツールです。`index.html` をブラウザで開くだけで動作します。

## 使い方

- `index.html` をブラウザ（Chrome / Edge 推奨）で開く
- マウスを動かすと波紋が広がり、クリックで大きな波紋が発生します
- 右上の⚙アイコンから設定パネルを開閉できます
  - **波の強さ**: 屈折の強さ（背景画像の歪み具合）
  - **波紋の減衰**: 波紋が収まる速さ
  - **自動の波**: チェックを入れるとランダムな位置に自動で波紋が発生
  - **背景画像を変更**: 任意の画像ファイルを選んで背景として表示
  - **波紋をリセット**: 現在の波紋をすべて消去

## 背景画像

`bg.jpg`（または `bg.png`）をこのフォルダに置くと、起動時に自動で読み込まれます。
画像が無い場合は青系のグラデーションが表示されます。

## ファイル構成

- `index.html` — ページ本体・設定パネルのUI
- `style.css` — レイアウト・パネルのスタイル
- `ripple.js` — 波紋シミュレーションと描画処理（WebGL / Canvas2D両対応）
- `bg.jpg` — 背景画像（任意）

## 技術メモ

- WebGLが使える環境では、フラグメントシェーダーで波の伝播をシミュレーションし、
  得られた高さマップの勾配で背景テクスチャのUVを屈折させています（jQuery Ripples系の手法を参考に独自実装）。
  シミュレーション解像度は最大480pxに抑えてパフォーマンスを確保しています。
- WebGLが使えない環境（ハードウェアアクセラレーション無効など）では、自動的にCanvas2Dの
  タイル分割方式にフォールバックします。画面をグリッド状のタイルに分け、各タイルを波の勾配に応じて
  わずかにずらして描画することで、同様の波紋効果を再現しています。
