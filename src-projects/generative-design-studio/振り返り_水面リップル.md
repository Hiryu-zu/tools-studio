# 振り返り — 水面リップル改善（konmari 風 桜×ミント水面）

最終更新: 2026-06-14
対象エフェクト: `src/effects/RippleEffect.ts`
背景画像: `public/water-bg.jpg`（ミント×桜のキラキラ水面、1672×941）

---

## 1. 目的

雛型から続く「マウスで波紋が広がる水面」を、参考（konmari 氏のデモ）のような
**桜×ミントのキラキラ水面**に引き上げる。スタジオの Three.js + ポストプロセス基盤を活用。

---

## 2. 改善前の問題

- スタジオ移植版は背景を**淡いプロシージャル生成**にしていたため、白っぽく平坦。
- 波紋・屈折・きらめきがほぼ見えず、深みも光の美しさも無かった。
- 生WebGLのため Bloom（発光）が無く、きらめきが光らない。

---

## 3. やったこと（Three.js + Bloom で作り直し）

1. **背景写真を再利用**: 最初に選んだ `bg.jpg` を `public/water-bg.jpg` にコピーして使用。
   一気に色の深みと桜の花びらが戻った。
2. **波シミュを half-float ピンポンに**: Three.js の `WebGLRenderTarget`（`HalfFloatType`）を2枚で
   ピンポンし、波動方程式を更新。8bit時代より高さ精度が上がり**滑らかな波紋**に。
   （x=高さ, y=速度 を生値で保持。8方向加重平均→速度更新→減衰→高さ更新→ドロップ加算）
3. **写真を屈折**: 波の勾配で背景UVをずらしてサンプリング＋薄い水色ティント。
   写真が濃いので明度の持ち上げは控えめに（白飛び回避）。
4. **星形(十字)きらめき＋スペキュラ**を付与し、**UnrealBloom** で発光。
5. **ACESトーンマッピング**で階調を整える。
6. 入力: マウス移動・クリックで波紋（ドラッグで連続）。自動波紋も任意でオン。

### パイプライン
```
[simScene quad: 更新シェーダー] → rtA/rtB ピンポン（half-float）
        ↓ (rtA.texture = 高さマップ)
[mainScene quad: 表示シェーダー = 屈折+ティント+スペキュラ+きらめき]
        ↓
EffectComposer: RenderPass → UnrealBloom → OutputPass
```

### 主要パラメータ（既定）
| パラメータ | 値 | 役割 |
|---|---|---|
| 屈折 refraction | 0.06 | 波で背景を歪ませる量 |
| 減衰 damping | 0.985 | 波の収まりやすさ |
| 輝き bloom strength | 0.6 | 発光の強さ |
| 露出 exposure | 0.9 | 全体の明るさ |
| Bloom threshold | **0.9（高め）** | 明るい所だけ発光（後述の白飛び対策） |
| sim解像度 | 最大512 | 波の精細さ/負荷のバランス |

---

## 4. ハマりどころ（重要）

- **背景写真が明るいと Bloom で全画面が白飛びする**。
  - 星景・イラスト版は背景が暗かったので Bloom 閾値 0.2〜0.3 でOKだった。
  - だが**明るいパステル写真**では、低い閾値だと写真全体が発光対象になり真っ白に。
  - 対策: **Bloom の threshold を 0.9 と高くし、きらめき/スペキュラ（値が1超になる部分）だけを発光**させる。
    あわせて露出を 0.9 に。これで写真の質感を保ったままキラキラだけ光る。
- 背景は `cover` フィット（画面とのアスペクト差を `uBgScale/uBgOffset` で吸収）。
- 半精度フロートRTは WebGL2 で利用可。高さを生値で持てるのでエンコード不要・高精度。

---

## 5. 操作・置き場所

- 背景差し替え: `public/water-bg.jpg` を置き換えるだけ（別の水面写真でも可）。
- 設定パネル: 屈折／減衰／輝き／露出／自動波紋。マウスで波紋、クリックで強い波紋。
- 差し替え後はリロード（キャッシュ時は Ctrl+Shift+R）。

---

## 6. 学び

- **明るい背景×Bloom は閾値が命**。暗い背景の設定を流用すると白飛びする。
- 既存の良い素材（bg.jpg）の再利用が、品質と工数の両面で最短だった。
- half-float のピンポンRTで、生WebGLより素直に・高精度に波シミュを書ける。
- 「光の美しさ」は Bloom＋スペキュラ＋星形きらめきの合わせ技で出る（星景と同じ発想）。
